# Hexalith.Applications.Tests

This repository contains unit tests for the Hexalith.Applications library, a component of the Hexalith platform.

## Overview

Hexalith.Applications.Tests provides comprehensive test coverage for all functionality within the Hexalith.Applications library, ensuring reliability and correctness of the implementation.

## Project Structure

The test project follows these conventions:

- Test files mirror the structure of the main project
- Each test class corresponds to a class in the main project
- Tests use the xUnit framework with Shouldly for assertions
